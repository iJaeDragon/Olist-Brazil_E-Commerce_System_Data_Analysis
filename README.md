# WebSocketOmok

## Chat GPT를 활용한 게임 설계 및 구현

해당 프로젝트는 게임을 만드는것에 대해 집중된 것이 아닌 Chat GPT를 적극 활용하여 결과물을 만들어내는 것에 집중된 프로젝트입니다.

그러며, 해당 결과물의 주제를 웹소켓기술을 사용한 오목게임을 만드는것으로 선정하였습니다.
